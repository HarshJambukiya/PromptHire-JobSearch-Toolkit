# 🚀 PromptHire – AI Job Search Toolkit

An AI-powered job search assistant built for students & freshers to find jobs smarter using prompt engineering.

---

## 🔥 Live Demo

👉 https://harshjambukiya.github.io/PromptHire-JobSearch-Toolkit/

---

## 💡 Features

* 🎯 AI Prompt Generator for Job Search
* 📧 Cold Email & LinkedIn DM Generator
* 🎤 Interview Preparation Prompts
* 📄 Resume Review Prompts
* 📊 Profile Strength Analyzer
* 🌐 Multi-platform job targeting (LinkedIn, Naukri, Internshala)

---

## 🧠 Tech Stack

* HTML, CSS, JavaScript
* Prompt Engineering (LLM-based logic)
* GitHub Pages (Free Deployment)

---

## 🎯 Use Case

This tool helps:

* Students find internships/jobs
* Freshers get their first job
* Job seekers automate job search using AI

---

## 🚀 How It Works

1. User enters profile details
2. System structures input
3. Generates optimized AI prompt
4. User pastes prompt into ChatGPT/Claude
5. Gets real-time job results
